Ejercicio De Java Taller 3
Enunciado 


Utilizando el ciclo que usted desee, crear un menú de ejecución infinita hasta que el
                usuario desee terminar dicho ciclo.

           Menu de Usuario
           1. Capturar nombre
           2. Saludar persona
           3. Salir del Sistema



                                                       
                                                    package Taller3.CicloMenu;
                                                    
                                                    import java.util.Scanner;
                                                    
                                                    /**
                                                        *
                                                        * @author Christian
                                                        */
                                                    public class CicloMenu {
                                                            public static void main(String[] args) {
                                                            int  control;
                                                            
                                                            do{
                                                                Scanner sc = new Scanner(System.in);
                                                                System.out.print("Menu Usuario"); 
                                                                System.out.print("1.Captura nombre"); 
                                                                System.out.print("2.Saludar persona"); 
                                                                System.out.print("3.Salir del sistema"); 
                                                                control = sc.nextInt();
                                                                }while(control<3);
                                                                        
                                                                    
                                                            
                                                                
                                                            } 
                                                    }
