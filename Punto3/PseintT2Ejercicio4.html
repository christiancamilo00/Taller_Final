Ejercicio De PseInt Taller 2
Enunciado


4. La video tienda que presta sus servicios de alquiler de películas a los usuarios del barrio el Porvenir, requiere de una aplicación que permita registrar el alquiler de las películas que adquieren sus usuarios. Para cada usuario se debe permitir la opción de alquilar película, consultar películas disponibles y recibir película en la video tienda con la opción de realizar anotaciones sobre estas si se llegan a presentar daños u otra novedad sobre la película. 

                                                        Proceso VideoTienda
                                                            Definir nompelicula Como Caracter;
                                                            Definir opcion Como Entero;
                                                            Definir pelicula1 Como Caracter;
                                                            Definir pelicula2 Como Caracter;
                                                            Definir pelicula3 Como Caracter;
                                                            Definir recpelicula Como Caracter;
                                                            
                                                            pelicula1 <- "Titanic";
                                                            pelicula2 <- "Spiderman";
                                                            pelicula3 <- "titanic";
                                                            
                                                            Escribir "Tienda Porvenir por favor elija una opcion";
                                                            Escribir "1. Alquilar pelicula";
                                                            Escribir "2. Consultar peliculas disponibles";
                                                            Escribir "3. Recibir pelicula";
                                                            Leer opcion;
                                                            
                                                            Segun opcion Hacer
                                                                1:
                                                                    Escribir "Como te llama la pelicula?";
                                                                    Leer  nompelicula;
                                                                2:
                                                                    Escribir "Las peliculas disponibles son: ";
                                                                    Escribir pelicula1;
                                                                    Escribir pelicula2;
                                                                    Escribir pelicula3;
                                                                    
                                                                3:
                                                                    Escribir "Por favor ingrese  las novedades de la pelicula";
                                                                    Leer recpelicula;
                                                            FinSegun
                                                            
                                                        FinProceso
